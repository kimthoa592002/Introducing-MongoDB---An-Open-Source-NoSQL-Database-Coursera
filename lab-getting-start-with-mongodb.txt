Getting start with MongoDB - Coursera Cloud IDE

Environment:
- Platform: Coursera / Cognitive Class
- Tool: Cloud IDE (browser-based)
- Database: MongoDB
- Access: MongoDB CLI (mongosh)

--------------------------------------------------
# 1. Find the version
db.version()

# 2. List databases
show dbs

# 3. Create database name training
use training

# 4. Create a collection
db.createCollection("mycollection")

# 5.  List collections
show collections

# 6. Insert documents into a collection
db.mycollection.insert({"color":"white","example":"milk"})
db.mycollection.insert({"color":"blue","example":"sky"})

# 7. Count the document in a collection
db.mycollection.countDocuments()

# 8. List all documents in a collection
db.mycollection.find()

# 9. Disconnect from the MongoDB server.
exit
